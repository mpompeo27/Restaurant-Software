### Restaurant Business Software Project

In the course Learn Intermediate Python3 course on Codecademy, the Function Arguments section of the course included a simple restaurant example for a fictional restaurant owned by our "friend" Jiho. 

I'm expanding that simple learning project into a functional piece of restaurant business software.

The basic functionality includes:
Assigning tables
Creating orders with unique order numbers
Adding food and drink items to an order
Adding reservations with unique reservation IDs, finding reservations by customer name and reservation time, and assigning tables from reservations
Calculating the bill for a table's order
Printing the bill either for a single payor or a split bill with multiple payors
Unassigning tables when customers have paid and leave the restaurant
Data persistence

This project will manage the development of further functionality necessary for the running of a restaurant, testing, changing the data persistence from a JSON file to SQL database, adding a UI, and transforming into a web app with Django.
